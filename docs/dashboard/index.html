How to read this

  • Composite is multi-objective. It prices all five axes (behavior, capability, coherence, safety, selectivity) plus an off-manifold geometry penalty — a method cannot win by sacrificing one axis. Negative composites (red) mean a coherence / off-shell blow-up.
  • Tiers matter. Every numeric cell carries n=X and a SCREENING (n≤3) or EVALUATION (n≥7) chip. Every row here is SCREENING (n=1) — candidate signal, not an external claim.
  • Safety is a hard gate. A non-zero CR (JailbreakBench compliance) is a leak and an automatic DISCARD regardless of behavior score.
  • Drill down. Runs are grouped by research campaign into separate tables. Click a header to sort that table; type in the filter box to search all rows. Each row links to its per-experiment page (and hypothesis page when resolvable). The starred gold row is the global champion.
Screening findings (n=1) — not external claims

Screening highlights S-1 … S-8

  • S-1 / E2. The naive expectation that bigger α monotonically helps is falsified: composite collapses past the α-cliff (E3-cliff goes from +0.83 at α=1 to −5.3e4 at α=24).
  • S-2 / N17, N5. Off-shell displacement tracks the composite collapse with R²≈0.81 across the α sweep.
  • S-3 / E27. A narrow cross-scale coherence window exists (small-rotation C3b & low-α add survive least-badly at L16).
  • S-4 / E4. diff-mean and PCA steering directions are near-collinear: cos≈0.994–0.996 on Qwen L21.
  • S-5 / N16. Angular displacement is an almost-perfect linear proxy for α (R²≈0.997).
  • S-6 … S-8. The cliff replicates across model scale (gemma-3-270m, gemma-3-1b) and operation (add least-bad, rotate / project_out worse); rotate at α≥1 is numerically explosive.
