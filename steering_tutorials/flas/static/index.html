FLAS  ·  flow-based activation steering

Lessons 2-3 pushed the residual stream with ONE one-shot edit — a fixed vector, then a learned rank-1 edit. FLAS instead learns a concept-conditioned velocity field v(h, t, c) and steers by integrating a flow from the unsteered activation to its steered position. The payoff: flow-time T is a continuous, zero-shot strength dial — drag it below to transport the activation less (or more) far along the SAME learned trajectory, with no retraining.

How to read this

  • Transport, not a jump. Instead of a single displacement, FLAS integrates h′ = h + ∫0T v(φt(h), t, c) dt in a few Euler steps — the activation flows to its steered position.
  • T is the dial. Integrate to a smaller T and you travel less far along the same path (gentler steer); larger T travels further. It is zero-shot: no per-strength retraining, unlike lesson 2's alpha.
  • One field, many concepts. The velocity is conditioned on a concept embedding c, so a single trained field steers toward any concept in the table (pick one from the dropdown).
  • Same gate, same judge. We reuse the lesson-1 harm probe as the conditional gate and the 1B self-judge (REFUSAL / COMPLIANCE / GIBBERISH). A 1B self-judge is weak — treat verdicts as illustrative, not publication-grade.
