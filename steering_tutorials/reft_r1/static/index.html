ReFT-r1  ·  lesson 3 dashboard

Lesson 2 re-installed refusal with a fixed diff-of-means vector — one direction, added with a hand-tuned strength. Lesson 3 swaps it for a learned rank-1 ReFT-r1 intervention (AxBench): a direction plus an affine readout trained end-to-end, so the edit is input-dependent and carries its own magnitude. Below: the learned edit running live, then its offline scores against the DiffMean vector and raw prompting — on both steering behaviour and concept detection.

How to read this

  • Learned, not hand-built. ReFT-r1 replaces the frozen vector with a rank-1 edit h' = h + r̂·((w·h + b) − (r̂·h)) trained by gradient descent. Because it reads h through w, the push is input-dependent — no external alpha to sweep.
  • Same gate, same judge. We reuse the lesson-1 harm probe as the conditional gate and the 1B self-judge (REFUSAL / COMPLIANCE / GIBBERISH), so the only thing that changed from lesson 2 is how the direction is produced.
  • Two jobs, one direction. AxBench scores a method on steering (does the edit install the behaviour cleanly?) and detection (does its readout separate the concept, as an AUC?). The offline panel reports both against DiffMean and prompting.
  • A 1B self-judge is weak. This is a teaching artifact: one small model plays writer, gate and judge. Treat verdicts as illustrative, not publication-grade.
