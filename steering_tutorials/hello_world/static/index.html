Safety Probe  ·  results dashboard

A 3-layer MLP reads one middle-layer activation vector from a frozen Gemma-1B and predicts harmful vs safe.

How to read this

  • The single-split headline is optimistic. With only ~200 examples, one train/test split gets lucky. The k-fold cross-validation numbers (with 95% CIs) are the trustworthy headline; trust those over the single split.
  • Out-of-domain calibration drifts. On a dataset the probe never trained on (XSTest), the ranking still transfers but the 0.5 threshold is miscalibrated — accuracy drops even while ROC-AUC stays high.
  • Small n means wide error bars. Every CV number carries a bootstrap/normal 95% CI. A gap smaller than the CI is not a real difference.
  • This is a teaching artifact, not a production safety filter. It shows that "is this harmful?" is linearly readable from activations — nothing more.
